This directory contains all datasets used in the paper.

⚠️ We use the terms **combination** and **blend** interchangeably, and **inspiration** and **analogy** interchangeably.

| Archive | 🤗 HuggingFace | Description                                                                                                                          |
|---------|-------------|--------------------------------------------------------------------------------------------------------------------------------------|
| `CHIMERA.zip` | [CHIMERA](https://huggingface.co/datasets/noystl/CHIMERA) | The CHIMERA knowledge base (~28K recombination edges)                                                                                |
| `recombination_prediction_data.zip` | [Recombination-Pred](https://huggingface.co/datasets/noystl/Recombination-Pred) | Train/valid/test splits for the recombination prediction task, alonf with additional task-specific information (e.g., leak detection) |
| `recombination_extraction_data.zip` | — | Annotated data for the recombination extraction task                                                                  |

`user_study_results.csv` contains the raw results from the human evaluation study.
